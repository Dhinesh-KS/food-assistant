## Technical Highlights

### Fast Search
< 50ms response time with multi-factor scoring:
- Name match: 10 points
- Description: 5 points
- Ingredients: 4 points
- Category: 3 points

### Free AI
Groq API (Llama 3.1 70B):
- Free tier
- < 2 second responses
- Good reasoning
- Production-ready

### Efficient State
Zustand + localStorage:
- Persistent cart
- Minimal re-renders
- Simple API

### Clean Architecture
```
app/        → Pages + API
components/ → React components
lib/        → Business logic
resources/  → Data + images
```
